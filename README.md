# TypeScript + Node.js Starter

###### Overview

A barebone TypeScript and Node.js project starter.

## Getting Started

###### Clone Repo

```
$ git clone https://github.com/mikejoyceio/typescript-node-starter.git
```

###### Install Dependencies

```
$ npm install
```

###### Run Build

```
$ npm run build
```
